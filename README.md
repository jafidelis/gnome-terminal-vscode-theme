# gnome-terminal-vscode-theme

Theme to Gnome Terminal like vscode

#### Tested in Ubuntu 20.04 LTS

##### Install theme
``git clone https://github.com/jafidelis/gnome-terminal-vscode-theme.git`

> cd gnome-terminal-vscode-theme

> ./install.sh

reopen terminal

##### Uninstall theme

> ./unistall.sh 

reopen terminal
